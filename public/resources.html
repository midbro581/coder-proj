Free Learning Resources

The best free books, courses, tools and references on the internet — handpicked and organized by topic.

Our Top 3 Recommendations if You're Just Starting

1. CS50 (cs50.harvard.edu) — the best first course in computing, free forever.  ·  2. freeCodeCamp (freecodecamp.org) — hands-on web dev curriculum, 300+ hours free.  ·  3. The Odin Project (theodinproject.com) — opinionated, project-based, full-stack.

Interactive Learning Platforms

freeCodeCamp
FREE
Full curriculum from zero to developer. HTML/CSS/JS, React, Python, Data Analysis, Machine Learning. 300+ hours. Earn real certificates.
freecodecamp.org★★★★★
The Odin Project
FREE
Best free full-stack curriculum. Project-based, opinionated path. Covers HTML, CSS, JavaScript, Node.js, React, Ruby on Rails. Community-backed.
theodinproject.com★★★★★
javascript.info
BOOK+
The best free JavaScript textbook online. Covers everything from basics to advanced patterns. Has interactive tasks at the end of every chapter.
javascript.info★★★★★
SQLZoo
FREE
Interactive SQL in the browser. Run real queries, see real results. Covers SELECT, JOIN, aggregates, subqueries. Best free SQL practice site.
sqlzoo.net★★★★☆
Flexbox Froggy
FREE
Learn CSS Flexbox by moving frogs to lily pads. 24 levels. Genuinely fun. You'll know every flexbox property by the time you finish.
flexboxfroggy.com★★★★★
CSS Grid Garden
FREE
Learn CSS Grid by watering a garden. 28 levels. Pair this with Flexbox Froggy and you'll master CSS layouts completely.
cssgridgarden.com★★★★★
Futurecoder
FREE
Interactive Python in the browser. Beginner-friendly, completely free. Great for absolute beginners who find other resources too fast.
futurecoder.io★★★★☆
SQLBolt
FREE
Interactive SQL lessons with exercises. Covers SELECT, WHERE, JOINs, aggregates, INSERT/UPDATE/DELETE. Clean and fast to get through.
sqlbolt.com★★★★☆

Free Video Courses

CS50x — Harvard
VIDEO
Harvard's Introduction to Computer Science. Free forever. C, Python, SQL, HTML/CSS/JS, Flask. Problem sets that will genuinely challenge you. The best first course.
cs50.harvard.edu/x★★★★★
CS50P — Python
VIDEO
Harvard's dedicated Python course. Free. Covers Python from basics to libraries, OOP, file I/O, regular expressions, and a final project.
cs50.harvard.edu/python★★★★★
Full Stack Open
FREE
University of Helsinki. Deep dive into React, Node.js, MongoDB, TypeScript, GraphQL. The best free full-stack course. University-level depth.
fullstackopen.com★★★★★
Traversy Media
VIDEO
Brad Traversy's YouTube channel. Crash courses on every web technology — HTML, CSS, JavaScript, React, Node, Python, and more. Practical and fast.
youtube.com/@TraversyMedia★★★★★
Fireship
VIDEO
100-second concept videos and deep dives. The best channel for getting a quick understanding of any technology. Dense, funny, informative.
youtube.com/@Fireship★★★★★
MIT Algorithms 6.006
VIDEO
MIT's Introduction to Algorithms — free lecture videos, assignments, and exams. The definitive algorithms course if you're aiming for Big Tech.
ocw.mit.edu★★★★★
Kaggle Learn
FREE
Free Python, Pandas, SQL, Machine Learning, and Data Visualization courses from Kaggle. Short, practical, with real exercises. Great for data science.
kaggle.com/learn★★★★☆

Free Books & Reading

Eloquent JavaScript
FREE BOOK
Marijn Haverbeke's book on JavaScript. Free online, purchase optional. Covers language fundamentals, browser DOM, and Node.js. One of the best programming books ever written.
eloquentjavascript.net★★★★★
Automate the Boring Stuff
FREE BOOK
Al Sweigart's practical Python book. Free to read online. Learn Python by automating real tasks — PDFs, spreadsheets, emails, web scraping, and more.
automatetheboringstuff.com★★★★★
You Don't Know JS
FREE BOOK
Kyle Simpson's deep-dive series on JavaScript internals. 6 books, all free on GitHub. Essential reading once you know the basics — explains how JS actually works.
github.com/getify/You-Dont-Know-JS★★★★★
Official Python Tutorial
OFFICIAL
Python.org's own tutorial. Surprisingly readable and comprehensive. Covers the entire standard library. The source of truth when you're not sure about something.
docs.python.org/3/tutorial★★★★☆
The Missing Semester (MIT)
FREE
MIT course on the tools nobody teaches in university — shell, Vim, Git, tmux, dotfiles, debugging, profiling. Every developer needs this.
missing.csail.mit.edu★★★★★
Invent With Python
FREE BOOK
Several free Python books by Al Sweigart — learn by building games, hacking ciphers, and making art. Great for beginners who want to build things immediately.
inventwithpython.com★★★★☆

Reference & Documentation

MDN Web Docs
REFERENCE
The gold standard reference for HTML, CSS, and JavaScript. Every web developer has this open every single day. If something is on MDN, it's correct.
developer.mozilla.org★★★★★
DevDocs.io
REFERENCE
One site with the documentation for every major language and framework — all searchable together. Works offline. Replaces 20 different documentation tabs.
devdocs.io★★★★★
Can I Use
REFERENCE
Does this CSS feature work in all browsers? This site answers that. Before using any new CSS or JS feature in production, check caniuse.com.
caniuse.com★★★★★
CSS-Tricks
GUIDES
Decades of CSS articles, guides, and almanac. The Flexbox and Grid guides here are legendary. Every CSS question you'll ever have is answered.
css-tricks.com★★★★★
Python Tutor
VISUALIZER
Step through your code and watch exactly what happens — which variables exist, what the call stack looks like. Incredible for debugging and understanding.
pythontutor.com★★★★★
Regex101
TOOL
Test regular expressions with real-time highlighting and explanation. Every regex pattern is explained in plain English. You'll use this for life.
regex101.com★★★★★

Practice & Projects

LeetCode
FREE TIER
The standard for coding interview prep. 2000+ problems. Start with Easy, progress to Medium. Aim for 100 problems before applying to big companies.
leetcode.com★★★★☆
Neetcode.io
FREE
Free roadmap of the most important LeetCode problems plus video solutions. Better than random LeetCode grinding — follow the roadmap systematically.
neetcode.io★★★★★
Frontend Mentor
FREE TIER
Real design mockups to build. Submit your solution and compare to others. The best way to build a frontend portfolio — you get a real brief and have to figure it out.
frontendmentor.io★★★★★
Codewars
FREE
Gamified coding challenges ("kata") in any language. 8-kyu (easy) to 1-kyu (very hard). Ranking system keeps you motivated. Great for Python and JavaScript practice.
codewars.com★★★★☆
VisuAlgo
FREE
Animated visualizations of sorting algorithms, data structures, and graph traversals. Watch Quicksort happen step by step. Essential for understanding algorithms.
visualgo.net★★★★★
First Contributions
FREE
A gentle guide to making your first open source contribution on GitHub. Teaches the fork → branch → PR workflow in a safe, friendly environment.
github.com/firstcontributions★★★★☆
